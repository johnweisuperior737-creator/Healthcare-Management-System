# Healthcare Management System

Healthcare Management System is a website which can control all the management related to a hospital, like admitting patients, booking beds, calling an ambulance, managing payments, creating reports of patients and many more thing online.

## Tech Stack

**Client:** 

- **React**
- **Redux Thunk**
- **Axios**
- **Ant-Designs**

**Server:**

- **Node Js**
- **Mongo DB**
- **Express Js**
- **JWT**
- **Nodemailer**

## Deployment

- **Client**  (Netlify)
- **Server** - (Cyclic)

## Features

- Admin controls
- Admitting Patients
- Booking beds and ambulance
- Creating appointments
- Generating reports 
- Overall control of hospital
